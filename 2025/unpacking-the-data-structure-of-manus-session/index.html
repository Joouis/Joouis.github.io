Unpacking the Data Structure of Manus Session

发表于 2025-03-21 分类于 ML/DL/LLM ， AI Agent 阅读时长 ≈ 15 分钟
Simple note of data interface of Manus session with examples.

The data comes from the case Comprehensive Tesla Stock Analysis and Investment Insights - manus.

The history data of a session is provided by the API https://api.manus.im/api/chat/getSession?sessionId=xxx.
