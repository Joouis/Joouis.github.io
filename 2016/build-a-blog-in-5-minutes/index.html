五分钟快速搭建一个博客

发表于 2016-04-09 分类于 笔记小抄 阅读时长 ≈ 2 分钟

作为部落格的第一篇文章，就记录一下它的诞生过程吧：）

博客很阳春，使用了 Github 和 Hexo，都是一谷歌就能找许多答案的技术。

搭建流程

  1. 安装 node.js 和 npm，推荐直接从官网下载源码编译；
  2. 安装 hexo：
  3. 安装主题，本博客使用 NexT 主题：
  4. 启用主题，在根目录下将_config.yml文件中的theme:改为你所需的主题名；
  5. 验证主题：
  6. Git 初始化并同步到 Github 上：
  7. 使用 hexo 部署功能，补充根目录_config.yml文件中的deploy，然后：
  8. 在 source 目录下加入CNAME文件，在登陆 DNS 服务商增添 CNAME 记录即可；
  9. 參考主題配置、第三方服务一步步客制化自己的博客。

心得

五分钟是完全足够搭建起博客的，不过真正费时的，就是一点点把它修改成自己想要的样子吧。 头一次使用 Hexo ，比 Jekyll 更方便，并且使用的 NexT 主题基于 Hexo 增添了很多功能接口，像搜索、评论平台等，只需要设置_config.yml一个文档基本上就能解决问题，确实很好用！

博客的用途是用来记录，Markdown 语言的特质就是帮助你专注在记录上，而今天使用的 Hexo 和 NexT 亦是如此。
